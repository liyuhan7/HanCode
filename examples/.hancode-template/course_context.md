# Course Context

## Course

AI4SE final project.

## Assignment

Build a lightweight Coding Agent Harness for student course projects.

## Demo Assignment

Implement a student grade statistics CLI project:

1. Read student grades from a CSV file.
2. Calculate average, highest, and lowest scores.
3. Support filtering by course.
4. Output statistics.
5. Write tests.
6. Generate README, TEST_REPORT, and KNOWLEDGE.

## Grading Rubric

- Harness mechanisms are deterministic code, not prompts.
- Phase gates prevent unsafe workflow shortcuts.
- MockLLM tests verify control flow without a real LLM.
- Trace and knowledge artifacts make the process reviewable.

## Submission Format

Final submission should include source code, tests, README, TEST_REPORT, KNOWLEDGE,
DELIVERABLES, and trace evidence.
